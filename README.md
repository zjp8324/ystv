# 📺 蜂蜜影视 / TVBox 配置

优化版蜂蜜影视/TVBox配置，参考uz影视的优秀源！

---

## 🎯 使用方法

在蜂蜜影视/TVBox的配置地址中输入：

```
https://raw.githubusercontent.com/zjp8324/ystv/main/tvbox.json
```

**jsdelivr加速地址（备用）：**
```
https://cdn.jsdelivr.net/gh/zjp8324/ystv@main/tvbox.json
```

## 📝 配置说明

### 网盘配置
在「配置中心」中可以配置以下网盘：
- 夸克网盘
- UC网盘
- 阿里云盘
- 115网盘
- 123网盘

### 百度网盘说明
由于技术限制，蜂蜜影视的pg.jar对百度网盘的支持有限。
- ✅ 搜索源：给力｜百度（聚合搜索百度网盘分享）
- ❌ 直接播放：不支持直接从百度网盘播放
