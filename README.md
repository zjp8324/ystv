# TVBox Spider - 蜂蜜影视接口

这是一个用于TVBox/蜂蜜影视的爬虫接口仓库，提供影视资源解析服务。

## 配置地址

在蜂蜜影视/TVBox中使用以下配置地址：

```
https://raw.githubusercontent.com/zjp8324/ystv/main/tvbox.json
```

或者使用jsdelivr加速：

```
https://cdn.jsdelivr.net/gh/zjp8324/ystv@main/tvbox.json
```

## 项目结构

```
ystv/
├── tvbox.json          # TVBox主配置文件
├── README.md          # 说明文档
└── spiders/           # 爬虫脚本目录
    └── example.js     # 示例爬虫
```

## 使用方法

1. 打开蜂蜜影视/TVBox
2. 进入设置 → 配置地址
3. 粘贴上面的配置地址
4. 返回首页即可使用
