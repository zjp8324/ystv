# TVBox Spider - 网盘资源接口

这是一个用于TVBox/蜂蜜影视的网盘爬虫接口仓库，提供各大网盘影视资源解析服务。

## 📋 配置地址

在蜂蜜影视/TVBox中使用以下配置地址：

**GitHub Raw:**
```
https://raw.githubusercontent.com/zjp8324/ystv/main/tvbox.json
```

**jsdelivr加速:**
```
https://cdn.jsdelivr.net/gh/zjp8324/ystv@main/tvbox.json
```

## 🎯 支持的网盘

- ☁️ 阿里云盘 - 支持Alist挂载
- 🌀 夸克网盘 - 高速网盘资源

## 🚀 使用方法

1. 打开蜂蜜影视/TVBox
2. 进入设置 → 配置地址
3. 粘贴上面的配置地址
4. 返回首页即可使用

## ⚙️ 配置说明

### 网盘Token配置

部分网盘需要配置Token才能正常使用：

1. 进入「配置中心」
2. 选择相应网盘
3. 按照提示获取并填入Token

### 阿里云盘配置

- 在配置中心添加阿里云盘Token
- 支持OpenToken和RefreshToken两种方式

### 夸克网盘配置

- 需要获取Cookie
- 支持夸克搜索和播放
